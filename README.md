# Roze Biohealth — Meta Ads Landing Page

A high-conversion landing page for Roze Biohealth's organic hydroxyapatite toothpaste line, built for UAE Meta ad traffic.

## Structure
- `index.html` — single-file static landing page (Tailwind via CDN, custom theme)
- `vercel.json` — Vercel deployment config

## Strategy
- **Angle**: luxury + toxin-free, anchored to Dr. David Roze's clinical credibility
- **Audience**: UAE biohackers to moms (broad)
- **CTA**: deep-links to existing rozebiohealth.com Woo product pages

## Conversion fixes vs. current Woo product page
- Strong above-the-fold promise (vs. generic product photo)
- Dr. Roze credibility moved from footer to hero & dedicated section
- 4 forced decisions reduced to 1 (pick a pack)
- Side-by-side comparison table (Roze vs. conventional)
- "Why hydroxyapatite" education section
- FAQ addressing every objection (kid-safe, fluoride water, tingling, etc.)
- Sticky mobile CTA (Meta traffic is ~85% mobile)
- 30-day guarantee + free UAE delivery banner
